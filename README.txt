##Prerequisite

#1: Visual Studio Code install on your machine!
#2: NodeJs install on your machine!
#3: MongoDB Install on your machine!
#4: MongoDB environment variable it set!


##How to run the Project.

Step #1 -> Open the project in Visual Studio Code.

Step #2 -> Open the terminal and write following command and hit Enter
	"npm install"

Step #3 -> And then write following command and hit Enter
	"node populate_db.js"
	(press Ctlr+c)

Step #5 -> And then write following command and hit Enter
	"npm start"

Step #6 -> Go to any Web-Brower and type "localhost:5000"
